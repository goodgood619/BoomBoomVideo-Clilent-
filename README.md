# BoomBoomVideo_Client

## 목적
동영상을 주로 보는 시청자들을 대상으로 하여, **동영상을 간편하게 등록 하거나 검색**하여 시청할 수 있는 **웹서비스**이다. 각각의 카테고리를 이용해 본인의 성향에 맞는 동영상을 시청하여, **현대인들의 스트레스 해소와 시간절약**을 목표로 하고있다. 

## 적용 기술
**JavaScript**, **Vue.js**
